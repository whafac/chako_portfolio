# 이미지 폴더

이 폴더에 포트폴리오에 사용할 이미지를 추가하세요.

## 필요한 이미지 파일

### 1. hero-image.jpg
- **위치**: 메인 페이지 (Hero Section)
- **용도**: 메인 페이지 중앙에 표시되는 인물 사진
- **권장 크기**: 600x800px 이상 (3:4 비율)
- **형식**: JPG, PNG, WebP

### 2. profile-image.jpg
- **위치**: About 페이지 프로필 섹션
- **용도**: 원형 배경 안에 표시되는 프로필 사진
- **권장 크기**: 400x400px 이상 (정사각형)
- **형식**: JPG, PNG, WebP

## 이미지 추가 방법

1. 이미지 파일을 이 폴더(`images/`)에 복사하세요
2. 파일명을 정확히 맞춰주세요:
   - `hero-image.jpg` (또는 .png, .webp)
   - `profile-image.jpg` (또는 .png, .webp)
3. 웹사이트를 새로고침하면 이미지가 자동으로 표시됩니다

## 이미지 최적화 팁

- 웹 성능을 위해 이미지 크기를 최적화하세요
- 온라인 도구 사용: [TinyPNG](https://tinypng.com/), [Squoosh](https://squoosh.app/)
- 파일 크기는 각각 500KB 이하로 권장합니다
